🔍 Social Media Analysis – OpenAI Followers Study

This project analyzes social media profile data with a specific focus on users who follow OpenAI(i took dummy data ).

The goal is to extract insights about these users, such as identifying highly active accounts and influential profiles based on followers and posting activity.

🎯 Objective

To analyze OpenAI followers and answer:

Who has the maximum number of posts?
Who has the highest number of followers?
Who follows the most accounts?
What types of user categories exist?
⚙️ Key Features
📄 Process raw unstructured data
🎯 Filter only users following OpenAI
🧹 Parse and clean profile data
📊 Perform targeted analysis on filtered users
💾 Store results in structured JSON format
🧠 Key Insight

Instead of analyzing all users, this project focuses on a target audience (OpenAI followers), making the analysis more relevant for:

Community insights
Influencer identification
Audience segmentation
🚀 Future Scope
Build follower network graph

Detect top AI influencers
